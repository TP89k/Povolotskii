# Инструкция по слиянию веток

## Как получить ветку с сервера
git fetch  # получить информацию о всех ветках
git checkout название_ветки  # переключиться в ветку

## Как слить одну ветку с другой
git checkout основная_ветка  # перейти в основную ветку
git merge ветка_для_слияния  # слить изменения

## Что сделать после слияния веток
1. Проверить конфликты (если есть)
2. Протестировать изменения
3. Отправить на сервер: git push
